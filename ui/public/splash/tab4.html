考核要求

 

3.  实验教学项目相关网络要求描述

3-1 有效链接网址

 http://cyemc.jmu.edu.cn/info/1151/1372.htm

3-2 网络条件要求 

（1）说明客户端到服务器的带宽要求（需提供测试带宽服务） 客户端外网宽带带宽≥5Mbps 为宜（带宽过小会导致加载时间过长）。

（2）说明能够提供的并发响应数量（需提供在线排队提示服务）； 可同时支持最大 200 个并发响应。

3-3 用户操作系统要求（如 Windows、Unix、IOS、Android 等） 

（1）计算机操作系统和版本要求

  支持 Windows、Linux、MacOS 等主流操作系统，推荐 Windows7 及以上版本；

  推荐使用 Google Chrome 浏览器或其他使用 Webkit 内核的浏览器；

  安装 Unity Web Player 浏览器插件 (不能禁用插件)。

（2）其它计算终端操作系统和版本要求 不支持移动终端。

3-4 用户非操作系统软件配置要求（如浏览器、特定软件等） 

（1）计算机非操作系统软件配置要求（需说明是否可提供相关软件下载服务）

 推荐使用 Google Chrome 浏览器或其他使用 Webkit 内核的浏览器，下载链接：http://cyemc.jmu.edu.cn/info/1151/1372.htm

（2）其它计算终端非操作系统软件配置要求（需说明是否可提供相关软件下载服务）

不支持

3-5 用户硬件配置要求（如主频、内存、显存、存储容量等）

 （1）计算机硬件配置要求

四核处理器、内存 8GB、显卡：GTX 1050 及以上同等计算能力，对硬盘没有要求。

（2）其它计算终端硬件配置要求

无

3-6 用户特殊外置硬件要求（如可穿戴设备等）

（1）计算机特殊外置硬件要求。

无

（2）无其它计算终端特殊外置硬件要求。

 无
